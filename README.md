# StealthChat

Децентрализованный защищённый мессенджер с анти-DPI обфускацией.

## Цель
Создать P2P-мессенджер, трафик которого неотличим от обычного HTTPS на порту 443.

## Ключевые особенности
- **Анти-DPI**: весь трафик через WSS (WebSocket Secure) на порту 443
- **E2E шифрование**: Double Ratchet + libsodium
- **Децентрализация**: нет центрального сервера
- **Анонимность**: регистрация по логин/пароль, без email/телефона
- **Onion-маршрутизация**: опциональная цепочка из 3 узлов

## Архитектура
```
Клиент (Flutter) 
  ↓ WSS/TLS 1.3 (порт 443)
Ретранслятор (опционально через цепочку)
  ↓ WSS/TLS 1.3 (порт 443)
Получатель (Flutter)
```

## Технологический стек
- **Flutter/Dart** - кроссплатформенный клиент
- **libsodium** - криптография (FFI)
- **web_socket_channel** - WSS клиент
- **sqflite_sqlcipher** - зашифрованная БД
